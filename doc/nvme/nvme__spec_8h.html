#define NVME_CLASS_CODE   0x10802

PCI class code for NVMe devices.

Base class code 01h: mass storage Subclass code 08h: non-volatile memory Programming interface 02h: NVM Express

#define NVME_DATASET_MANAGEMENT_MAX_RANGES   256

Indicates the maximum number of range sets that may be specified in the dataset mangement command.

#define NVME_GLOBAL_NAMESPACE_TAG   ((uint32_t)0xFFFFFFFF)

Use to mark a command to apply to all namespaces, or to retrieve global log pages.
