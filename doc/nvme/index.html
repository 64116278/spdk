SPDK Userspace NVMe Driver

Public Interface

  • nvme.h

Key Functions

  • nvme_probe() Enumerate the NVMe devices attached to the system and attach the userspace NVMe driver to them if desired.
  • nvme_ns_cmd_read() Submits a read I/O to the specified NVMe namespace.
  • nvme_ns_cmd_write() Submits a write I/O to the specified NVMe namespace.
  • nvme_ns_cmd_deallocate() Submits a deallocation request to the specified NVMe namespace.
  • nvme_ns_cmd_flush() Submits a flush request to the specified NVMe namespace.
  • nvme_ctrlr_process_io_completions() Process any outstanding completions for I/O submitted on the current thread.
