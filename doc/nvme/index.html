SPDK Userspace NVMe Driver

Public Interface

  • spdk/nvme.h

Key Functions

  • spdk_nvme_probe() Enumerate the NVMe devices attached to the system and attach the userspace NVMe driver to them if desired.
  • spdk_nvme_ns_cmd_read() Submits a read I/O to the specified NVMe namespace.
  • spdk_nvme_ns_cmd_write() Submits a write I/O to the specified NVMe namespace.
  • spdk_nvme_ns_cmd_deallocate() Submits a deallocation request to the specified NVMe namespace.
  • spdk_nvme_ns_cmd_flush() Submits a flush request to the specified NVMe namespace.
  • spdk_nvme_qpair_process_completions() Checks for and processes completions on the specified qpair.
